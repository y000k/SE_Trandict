## 2. 引用

词库来自于[english-words](https://github.com/dwyl/english-words)和[dict](https://github.com/kajweb/dict), 过滤了包含符号和数字的条目。

## 3. 技术栈

数据库 > Sqlite

数据处理 / 框架 > Python3.7

界面 > Qt (Pyside2)
